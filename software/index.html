Software

Pacotes R

pollingBR

O pollingBR é um pacote em R, criado para facilitar a utilização de dados contidos no banco de dados de pesquisas eleitorais do site Poder360. O pacote extrai dados da API deste repositório, desenvolvida pelo Volt Data Lab. Este pacote foi produzido por mim com contribuições de Sérgio Spagnuolo, Denisson Silva e Fernando Meireles.

Para download e instruções de uso, acesse o repositório do pollingBR no GitHub.

crossfire

O crossfire é um pacote em R utilizado para extrair dados da API de dados abertos do projeto Fogo Cruzado, desenvolvida pelo Volt Data Lab. O pacote foi desenvolvido por mim, com contribuições de Sérgio Spagnuolo e Denisson Silva.

O Fogo Cruzado é “uma plataforma digital colaborativa que tem o objetivo de registrar a incidência de tiroteios e a prevalência de violência armada na região metropolitana do Rio de Janeiro e de Recife”.

Para download e instruções de uso, acesse o repositório do crossfire no GitHub.

newsatlasbr

O newsatlasbr é um pacote em R destinado a extrair dados da API de dados abertos do Atlas da Notícia, projeto que mapeia veículos de jornalismo no Brasil. Desenvolvi o pacote, com contribuições de Sérgio Spagnuolo e Renata Hirota.

Para download e instruções de uso, acesse o repositório do newsatlasbr no GitHub.
